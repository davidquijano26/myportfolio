
# My Portfolio

This repository contains my personal portfolio website developed as part of my academic and professional preparation as a Bachelor of Science in Information Technology student, with a focus on networking and IT support.

The portfolio showcases my basic technical skills, online certifications, and a deployed web project that demonstrates foundational experience in system deployment and cloud-based hosting.

## Technologies Used
- HTML5  
- CSS3  
- GitHub (Version Control)  
- Netlify (Cloud Hosting & Deployment)

---

##  Deployment
The portfolio website is deployed using **Netlify**, integrated directly with this GitHub repository. Continuous deployment is enabled, allowing automatic updates whenever changes are committed to the main branch.

The website is publicly accessible through a Netlify-generated domain and is served over HTTPS.
